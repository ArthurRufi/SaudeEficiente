# SaudeEficiente
Projeto destinado a facilitação e produtividade em unidades de saúde..

## Objetivo

Classificar para profissionais da saude e postos de atendimento um projeto de codigo aberto que venha a resolver alguns problemas úteis no dia a dia que possam ter passado batido e outros que já uma queixa a anos.

## Funcionalidades

### Sistema de Gestão
O sistema de gestão para o "Saude Eficiente" conta com uma serie de beneficios tanto para os profissionais como para os usuarios.

#### Para o Usuario
- Disponibilidade de Medicos
- Disponibilidade de vacinação
- Receita Online]
- Dicas
- Mapeamento para agentes de saude
- Assistencia social

#### Para o medico
- Prontuario digital de pacientes baseado no CPF
- Lista de medicação
- Resumo inteligente (Usando AI)
- Lista atencipada da fila de espera
- Receituario com AI

#### Para o enfermeiro
- Receitas Online do paciente
- Lista de medicação com nome do medico responsavel
- Tabelas necessarias
- Guia SUS

#### Para o assistente social
- Informações do Cidadão
- Registro familiar
- Lista de confiança
- Visitas


#### Para o Psicologo
- Informações particulares do paciente
- Historico de transtornos
- Resumo
