# Statistic

## Função

Determinar de forma direta a previsibilidade de compra de um paciente para conseguir ver o que ele vai comprar como e quando vai ser essa compra, fazendo assim com que farmacias modelem como esse paciente vai comprar
No geral tudo que for para o setor de estatisca para pacientes vai cair aqui.


## Exemplo

Grafico no mapa de quais pacientes compram um remedio X de pressão ou Y de diabetes.

# Arquitetura

Tabela *compras* para determinar as compras daquele pacientes com filtro de tempo, tabela *previsão* para calcular tempo e possibilidade de compra de produto para cada data.
