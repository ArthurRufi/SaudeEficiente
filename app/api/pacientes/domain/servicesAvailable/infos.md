# Services Available

## Função
Recber todos os serviços disponivel para aquele usuario como medicos, postos, vacinação, calendario, campanha e outras informações ainda adicionar.
A Services Available vai tratar para o paciente somente suas infos basicas nos horarios necessarios, por exemplo, vai ser servir como uma dupla confirmação de agendamento, com codigo de consulta que vai ser gerado tanto para o paciente como para o psf ou clinica.


## Arquitetura de banco de dados

Tabela *Informações* com as informações do paciente, tabela *historicoclinico* que vai detalhar todo o historico clinico do paciente, tabela *locais* que vai pegar todos os locais de consulta e agendamentos do paciente.
